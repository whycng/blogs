在 VS Code 里，熟练使用快捷键能让你写代码的效率提升一个档次，甚至可以彻底摆脱鼠标。以下是为你整理的最常用“神级”快捷键清单：

### 1. 窗口与界面控制（你的核心需求）

这一组快捷键可以帮你快速切换视图，腾出代码编辑空间。

* **显示/隐藏侧边栏**：`Ctrl + B` (Windows/Linux) / `Cmd + B` (Mac)
* **唤出/隐藏集成终端**：**`Ctrl + ``** (反引号，Tab键上方)
* **显示所有视图（资源管理器、搜索、Git等）**：
* 资源管理器 (Explorer)：`Ctrl + Shift + E`
* 全局搜索 (Search)：`Ctrl + Shift + F`
* 运行与调试 (Debug)：`Ctrl + Shift + D`


* **全屏模式**：`F11`
* **拆分编辑器（分屏）**：`Ctrl + \` (反斜杠)

---

### 2. 代码编辑与文件导航

练习 LeetCode 或写项目时，这几个键能省去大量翻页时间。

* **快速查找文件**：**`Ctrl + P`** (输入文件名直接跳转，不需要用鼠标点文件夹)
* **快速搜索符号/函数**：`Ctrl + Shift + O`
* **查看定义（预览）**：**`Alt + F12`** (在当前位置弹出一个小窗看代码定义，不用跳走)
* **跳转到定义**：`F12`
* **全局重命名**：`F2` (比如你想把 `root` 改成 `node`，它会把整个项目里相关的都改了)
* **代码格式化**：`Alt + Shift + F`

---

### 3. 多行编辑（效率神器）

* **多光标选中**：`Alt + 鼠标点击` (可以在多个地方同时输入)
* **向上/下复制当前行**：`Alt + Shift + ↑/↓`
* **向上/下移动当前行**：`Alt + ↑/↓`
* **快速注释/取消注释**：`Ctrl + /`

---

### 4. 命令面板

这是 VS Code 的**灵魂**。如果你忘了某个快捷键，或者想执行某个插件的功能，按这个就对了：

* **命令面板**：**`Ctrl + Shift + P`** (输入任何你想做的操作，比如 "Markdown Preview"、"Restart Extension Host" 等)

---

### 5. 针对你提到的“预览”

* **Markdown 预览**：如果你在写 `.md` 文档，按 `Ctrl + Shift + V`。
* **HTML/其他预览**：VS Code 原生不带 HTML 实时预览，建议去插件商店搜 **"Live Server"**。安装后，右键 HTML 文件点 "Open with Live Server" 即可。

### 💡 小贴士

你可以随时通过菜单栏的 **文件 (File) -> 首选项 (Preferences) -> 键盘快捷方式 (Keyboard Shortcuts)** 来查看或修改所有的键位映射。
